Caelestia Local Installation Packages

Pre-built packages for local installation to ~/.local by Rust-Frog

Usage

Install via the dotfiles installer:

git clone https://github.com/Rust-Frog/dots.git
cd dots
./install.fish

Manual Installation

# Download and extract to ~/.local
cd ~/.local
curl -L https://rust-frog.github.io/dots-package/caelestia-cli-latest.tar.gz | tar xz
curl -L https://rust-frog.github.io/dots-package/caelestia-shell-latest.tar.gz | tar xz

# Add ~/.local/bin to PATH if not already
export PATH="$HOME/.local/bin:$PATH"

Packages

  • caelestia-cli - CLI tool for wallpaper, color schemes, shell control
  • caelestia-shell - QML desktop shell with C++ plugin

Installation Locations

  • ~/.local/bin/caelestia - CLI binary
  • ~/.local/lib/qt6/qml/Caelestia/ - QML plugin
  • ~/.local/share/quickshell/caelestia/ - Shell QML files
  • ~/.local/lib/caelestia/ - Shell extras
